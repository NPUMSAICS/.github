## 📖 关于我们 | About Us

**西工大附中人工智能与计算机科学社团**是在校团委、政教处领导下，由学生会学习部直接指导，汇集全校热爱人工智能与计算机科学的学生自愿组成的学术科技类学生团体。

本 GitHub Organization 是社团的**共创空间（Co-Creation Space）**，是我们存放开源项目、学习资料和竞赛代码的数字化阵地。

* **社团宗旨**：搭建学习交流、创新实践、竞赛辅导、技术服务于一体的综合性平台。
* **培养目标**：通过系统化的培训和项目式的实践，提升会员的科学素养和工程能力。
* **愿景展望**：着眼未来科技趋势，培养具备前瞻性视野和创新性思维的复合型人才。

---

## 🛠 我们在做什么 | What We Do

### 1. 开源项目实践 (Open Source)
我们鼓励会员通过 GitHub 进行协作，开发实用工具、校园服务应用或学术研究模型。在这里，你可以体验到从 `git commit` 到 `Code Review` 的完整工作流。

### 2. 算法与竞赛 (Competitive Programming & AI)
* **AI Models**: 涵盖深度学习、计算机视觉（CV）及自然语言处理（NLP）的复现项目。
* **数学建模**: 运用 Python/Rust 等工具进行数据分析与数学仿真。

### 3. 技术沙龙 (Tech Talks)
定期分享前沿技术，所有讲义与 Demo 均在仓库中公开。
